🛒 DP-Store - MVP Summary

A full-stack e-commerce app built in 5 days

📌 Project Overview

DP-Store is a modern full-stack e-commerce MVP inspired by eBay. It includes a user-facing shopping frontend, a robust Express backend API, and a powerful admin dashboard for managing users, products, and orders.

⚙️ Tech Stack

  • Frontend: Next.js + Tailwind CSS + Zustand
  • Admin Panel: Next.js + Zustand + React Hook Form
  • Backend: Express.js + TypeScript + Firebase
  • Database: Firebase Firestore
  • Image Upload: Cloudinary
  • Auth: Firebase (Google, GitHub, Email)
  • Deployment: Vercel (frontend), Google Cloud (backend)
  • Tools: Docker, Railway, FileSaver, CSV Export

✅ Features

User Features
  • 🛍️ Browse products by category
  • 💖 Wishlist & 🛒 Cart functionality
  • 🔐 Auth via Google, GitHub, Email
Admin Features
  • 📦 Product CRUD with Cloudinary images
  • 👥 User management (promote/delete)
  • 🧾 CSV export, search, pagination
  • 🪵 Activity log in Firestore

🚀 Deployment Journey

  • ✅ Local testing of all modules (frontend + backend)
  • ⚠️ Railway: CI/CD failures and broken URL issues
  • ✅ Dockerized backend locally for testing
  • ✅ Final deployment on Google Cloud Compute with static IP + firewall config
  • ✅ Frontend + Admin hosted via Vercel

🗓️ 5-Day MVP Timeline

  • Day 1: Project setup, Firebase Auth, planning, rough documentation
  • Day 2–3: Full frontend, backend, admin dashboard, product CRUD
  • Day 4: UI/UX fixes, toast feedback, Cloudinary integration
  • Day 5: Docker testing, deployment attempts, final deploy to Google Cloud
